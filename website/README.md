# MinfoAI 2.0 Website

Questo sito web è stato creato per presentare il bot MinfoAI 2.0 e per essere utilizzato con strumenti di analisi come "Google Pomelli".

## Come usare

1.  **Deployment Rapido:**
    *   Trascina la cartella `website` su servizi come [Netlify Drop](https://app.netlify.com/drop) per metterlo online in pochi secondi.
    *   Oppure usa Vercel o GitHub Pages.

2.  **Google Pomelli:**
    *   Una volta che il sito è online (avrai un URL tipo `https://tuo-sito.netlify.app`), inserisci questo URL nella schermata di "Google Pomelli".
    *   L'AI analizzerà il contenuto del sito (che descrive le partnership, l'AI, i tier) e genererà il tuo "Business DNA".

## Struttura
*   `index.html`: La pagina principale.
*   `style.css`: Stili moderni e premium.
*   `script.js`: Animazioni semplici.
